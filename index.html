2 安装

2.1 从官网下载redis环境

        从官网上找到资源，直接使用wget的方式下载

2.2 解压

         进入下载目录，解压下载的redis安装包

2.3 编译

        进入解压目录，执行编译指令make MALLOC=libc

         在安装目录执行make install进行安装

        在安装目录执行./utils/install_server.sh

3 查看、启动、停止

3.1 查看

        通过ps -ef|grep redis命令查看redis进程

3.2 启动

        开启Redis服务操作通过/etc/init.d/redis_6379 start命令，也可通过（service redis_6379 start）

3.3 停止

        关闭Redis服务操作通过/etc/init.d/redis_6379 stop命令，也可通过（service redis_6379 stop）

Elasticsearch学习笔记(三)—在centos7上搭建环境

发表于 2018-12-28 | 更新于 2019-02-22 | 分类于 大数据 ， Elasticsearch

1 系统环境

  • 操作系统：CentOS Linux release 7.5.1804 (Core)

  • Java 1.8.0_181

2 Elastic安装

2.1 安装的简单说明

按照官网的说明，安装步骤如下：

  1. 下载和解压安装包

  2. 启动程序

  3. 检查是否启动成功

2.2 下载安装包

打开官网，点击【下载】，选择Elasticsearch，再选择ZIP安装包

2.3启动服务

  1. 解压tar -xvf elasticsearch-6.5.4.tar.gz

  2. 进入下载的目录$ cd elasticsearch-6.5.4/bin

  3. 执行启动指令$ ./elasticsearch

2.4 获取版本信息

  1. 另开一个标签页

  2. 执行指令$ curl localhost:9200

    控制台打印信息如下：

2.5 备注

  • 停止Elastic，在启动选项卡中直接control+c即可停止。

  • 如果你想把 Elasticsearch 作为一个守护进程在后台运行，那么可以在后面添加参数 -d。

  • 默认情况下，Elastics只允许本机访问，如果需要远程访问，可以修改 Elastic 安装目录的config/elasticsearch.yml文件，去掉network.host的注释，将它的值改成0.0.0.0，然后重新启动 Elastic。

  • 安装中文插件

    在安装目录下，运行指令：$ ./elasticsearch-plugin install https://github.com/medcl/elasticsearch-analysis-ik/releases/download/v6.5.4/elasticsearch-analysis-ik-6.5.4.zip。

    注意版本要一致。

    安装完成后，重启。
