Elasticsearch学习笔记(二)—在macOS上搭建环境

发表于 2018-12-28 | 分类于 工作技能 ， 大数据 ， Elasticsearch

1 系统环境

  • 操作系统：Mac OS X(10.13.2 )

  • Java 1.8.0_151

2 Elastic安装

2.1 安装的简单说明

按照官网的说明，安装步骤如下：

  1. 下载安装包

  2. 启动程序

  3. 检查是否启动成功

2.2 下载安装包

  • 方式一

    打开官网，点击【下载】，选择Elasticsearch，再选择ZIP安装包

  • 方式二

    输入指令$ curl -O https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-6.1.2.zip，控制台打印日志如下：

2.3启动服务

  1. 进入下载的目录$ cd elasticsearch-6.1.2

  2. 执行启动指令$ ./bin/elasticsearch

    控制台打印信息如下图：

2.4 获取版本信息

  1. 另开一个标签页

  2. 执行指令$ curl localhost:9200

    控制台打印信息如下：

2.5 备注

  • 停止Elastic，在启动选项卡中直接control+c即可停止。

  • 如果你想把 Elasticsearch 作为一个守护进程在后台运行，那么可以在后面添加参数 -d。

  • 默认情况下，Elastics只允许本机访问，如果需要远程访问，可以修改 Elastic 安装目录的config/elasticsearch.yml文件，去掉network.host的注释，将它的值改成0.0.0.0，然后重新启动 Elastic。

  • 安装中文插件

    在安装目录下，运行指令：$ ./bin/elasticsearch-plugin install https://github.com/medcl/elasticsearch-analysis-ik/releases/download/v6.1.2/elasticsearch-analysis-ik-6.1.2.zip。

    注意版本要一致。

    安装完成后，重启。

3 Kibana安装

3.1 安装的简单说明

按照官网给出的资料（链接），安装步骤顺序如下：

  1. 下载Kibana安装包

  2. 打开config/kibana.yml配置文件，并进行设置

  3. 运行指令启动Kibana

  4. 打开浏览器访问http://localhost:5601

3.2 下载安装包

  • 方式一：

    打开官网，点击【下载】，选择Kibana，再选择ZIP安装包（链接）。

  • 方式二：

    输入指令$ wget https://artifacts.elastic.co/downloads/kibana/kibana-6.1.2-darwin-x86_64.tar.gz

3.3 修改配置文件

  1. 将下载的安装包进行解压$ tar -zvxf kibana-6.1.2-darwin-x86_64.tar。

  2. 进入解压后目录的config目录下$ cd kibana-6.1.2-darwin-x86_64/config/。

  3. 找到kibana.yml文件，进行编辑$ vim kibana.yml。

  4. 找到elasticsearch.url，去掉前面的注释，并且将值设置为Elasticsearch的实例。更改如下图

3.4 启动服务

进入安装目录的config文件夹下，执行指令$ ./kibana。

3.5 打开浏览器访问

打开浏览器，输入地址http://localhost:5601。
